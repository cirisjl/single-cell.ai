## Search and Download Benchmark Dataset

Users can search datasets using filters such as **species**, **category**, **author**, **organ part**, **cell type**, and **disease status**.

You can:

* **Open the Benchmark Details** page to view metadata and **download** the dataset directly.
* Or, **download via the OSCB Python API** using the **Benchmark ID**:

<div align="center">
<img src="/images/doc/view_benchmarks.png" width="80%">
</div>

> [!TIP]  
> Click the **column headers** to **sort** the table. 

On the **Benchmark Details** page, you can view:

* **Dataset metadata** — information about the dataset used in the benchmark.
* **Task info** — details such as task type, parameters, and selected methods.
* **Performance plots** — visualizations including bar and line charts for evaluating accuracy, efficiency, and scalability.
